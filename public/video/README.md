# Video Background for Hero Section

## Required Video File

Place your video file here and name it: **videoo.mp4**

### Video specifications:
- **Format**: MP4 (H.264 codec recommended for broad compatibility)
- **Resolution**: 1920x1080 (Full HD) minimum, 2560x1440 or higher recommended
- **Aspect ratio**: 16:9
- **Duration**: 10-30 seconds (will loop)
- **File size**: Keep under 5MB for optimal page load
- **Content**: Abstract tech/electronics themed video with dark tones (high black as mentioned)
  - Ideas: Electronics close-ups, futuristic animations, glowing circuits, tech devices in motion
  - Should have dark/black dominant colors for text readability

### Important notes:
- Video must be **muted** to enable autoplay on all browsers
- `autoPlay` attribute requires `muted` attribute on most browsers
- `playsInline` ensures mobile devices play inline instead of fullscreen

### Optional: Poster image
Add a fallback image named `poster.jpg` (same folder) that displays while video loads or if video fails to load.
Poster should be a high-quality still frame from the video or a dark tech-themed image.

## Example video settings for rendering:
- Bitrate: 2-4 Mbps
- Frame rate: 24-30 fps
- Use two-pass encoding for better quality
- Consider using a CDN for hosting if file size exceeds 10MB
