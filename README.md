# JP Tech — Electronics Store Rwanda

A full-featured e-commerce platform for selling electronics across Rwanda, built with Next.js 16, MongoDB, and Tailwind CSS.

## 🚀 Features

### Customer-Facing
- **Homepage**: Vibrant marketplace with featured deals, categories, and trust sections
- **Bilingual Support**: English & Kinyarwanda (toggle in header)
- **Product Catalog**: 9 categories, 100+ products with filters & search
- **Product Details**: Image gallery, specifications, pricing in RWF
- **Shopping Cart**: Persisted locally, cart drawer & full cart page
- **Checkout Flow**: Multi-step checkout with Rwanda district delivery options
- **Order Tracking**: Check order status and delivery details

### Admin Features
- **Admin Login**: Secure login (Demo: admin@jptech.rw / JPTech2024!)
- **Dashboard**: Real-time stats, revenue, low-stock alerts
- **Product Management**: Add, edit, delete products with images
- **Category Management**: Organize products by category
- **Order Management**: View and process customer orders

## 🎨 Design Direction

**Color Scheme**:
- Beige (#F5F5DC) - 50% (backgrounds)
- Black (#000000) - 30% (text, accents)
- Gold (#D4AF37) - 20% (highlights, buttons)

## 📦 Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript, React 19
- **Styling**: Tailwind CSS with custom theme
- **Database**: MongoDB Atlas

## 🚀 Getting Started

```bash
npm install
npm run dev
```
